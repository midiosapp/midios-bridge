# midiOS Bridge

macOS bridge for midiOS. Enables real-time communication between iPhone and Ableton Live.

---

## What is midiOS?

midiOS is a gesture-driven performance instrument for iPhone.

It allows musicians to control harmony, expression, and DAWs in real time using touch, tilt, and motion.

Works with any DAW. Built for Ableton Live.

---

## What is the Bridge?

The midiOS Bridge connects the iPhone app to your Mac, enabling:

- Real-time tempo and phase sync
- Two-way communication with Ableton Live
- Clip state awareness and performance feedback
- Stable connection between midiOS and your DAW

---

## System Architecture

```
iPhone (midiOS app)
    ↓ USB MIDI
Mac (midiOS Bridge)
    ↓
Ableton Live (Remote Script)
```

The Bridge acts as the central communication layer between midiOS and Ableton Live.
